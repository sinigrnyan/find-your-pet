Кликни по карте
Как пользоваться:

1. Клик по карте — выбор точки
2. В точке можно добавить наблюдение
или нажать кнопку"построить" для построения маршрутов поиска
3. Клик по наблюдениям — посмотреть сообщение/фото
4. Сохраняй уже построенные маршруты и делись ID с другими волонтёрам
